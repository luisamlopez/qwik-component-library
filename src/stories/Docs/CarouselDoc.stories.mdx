# Carousel

Para usar este componente, debes importarlo desde:

```
../components/Carousel/Carousel
```

El carousel, para poder mostrarse en desktop y mobile, se debe utilizar con CarouselLarge y CarouselSmall.

### Props

- `slides: object array` - Array de objetos con las propiedades:
  - `name: string`
  - `image:  string`
  - `phrase:  string`
  - `short_description: string`
  - `link:  string`

### Ejemplo de uso

```

import Button from "../"components/Button/Button";

function App() {
  return (
    <div>
      <Carousel slides={slides} />
    </div>
  );
}

export default App;

```

- Se utiliza el link con un id o con una URL.
